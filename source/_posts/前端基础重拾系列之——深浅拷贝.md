### 深拷贝

> deepClone

解决[浅拷贝](#浅拷贝)拷贝对象的时候, 对象中存在深层的嵌套对象的问题. 例如:

```ts
const obj = {
  b: 1,
  c: 2,
  d: {
    e: 3,
    f: [],
    g: {
      h: {
        ...
      },
    },
  },
};
```

这时, 使用[浅拷贝](#浅拷贝)会出现意想不到的情况 —— `d`键的地址会被clone. 而`深拷贝`则能解决这个问题.
