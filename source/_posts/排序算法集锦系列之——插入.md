也作`直接插入`排序, 最稳定的算法, 来记录一下


<!-- more -->


## 时间复杂度

------

O(n ^ 2)

## 空间复杂度

------

O(1)

## 思路

------

- 外层遍历样本数
- 内层遍历已排序数组
  - 未符合条件, 进位处理
  - 符合条件, 插入

## 源码

------

```js
function insertSort(nums) {
  for(let i = 1; i < nums.length; i++) {
    let j = i - 1;
    let temp = nums[i];

    // 符合条件
    while(j >= 0 && nums[j] > temp) {
      nums[j + 1] = nums[j];
      j--;
    }
    // 不符合条件
    nums[j + 1] = temp;
  }

  return nums;
}
```
