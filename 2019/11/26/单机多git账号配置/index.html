    单机多git账号配置

    由于需要在公司的电脑上登录不同的 Git 服务器:

    • 公司的 Gitlab
    • 自己的 Github

    两者使用不同的邮箱, 所以需要在一台电脑上配置不同的 Git, 特此记录完整的配置过程.

    错误捕捉

    ssh: Could not resolve hostname xxx: Name or service not known

    产生原因

    ~/.ssh/config 中的 Host 和 HostName 不相等.

    解决办法

    将两者设为相同的地址即可.
