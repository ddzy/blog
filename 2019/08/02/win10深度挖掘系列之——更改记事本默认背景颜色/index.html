    win10深度挖掘系列之——更改记事本默认背景颜色

    win10记事本的背景默认为白色, 看久了有些刺眼, 所以萌生出将其更改为护眼的绿色, 将修改的步骤记录于此.

    更新

    [2019-8-2]

    • Initial release

    步骤

    • Win + R打开运行窗口, 输入regedit

    • 打开HKEY_CURRENT_USER/Control Panel/Colors

    • 将Window的值改为任何想要的颜色

    • 重启电脑
