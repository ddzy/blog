    3.1 前置知识

    经过上一次的Route的源码学习, 我们学习到了Route组件的设计思路, 即:

    Route的作用主要是计算match, 根据match和决定渲染component, 那么这篇文章学的Switch, 则是:

    PS: 计算match, 将计算之后的computedMatch传递给Route

    PS: Switch的子组件Route只能有一个被渲染出来
